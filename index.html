Sharpexpressions

Expression evaluator for C# using .NET framework 3.5 that doesn't require JIT compilation

View on GitHub Download .zip Download .tar.gz

Welcome to the SharpExpression page.

SharpExpressions is an open-source library that parses an expression and evaluates it during the runtime. It doesn't use Linq expressions or code emission, making it a good fit for .NET runtimes that require AOT compilation, such as Mono in iOS.
